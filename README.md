# My Vagrant Setup

1. `$ vagrant up`
1. `$ vagrant ssh`
1. Remove html folder from document root in `/etc/apache2/sites-enabled/000-default.conf`
1. Create your virtual hosts (don't forget to edit your own hosts file!)
1. Optional: Add `cd /vagrant` to `/etc/bash.bashrc`
